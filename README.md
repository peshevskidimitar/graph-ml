# Graph Machine Learning for Web-Based Systems

## Requirements

The required Python version is `3.10.10`.

```bash
python3 --version
```

## Setup Instructions

1. Clone the repository and navigate to the project's directory.

```bash
git clone https://github.com/peshevskidimitar/graph-ml
cd graph-ml
```

2. Create and activate a virtual environment.

```bash
python3 -m venv .venv
source .venv/bin/activate
```

3. Install the project dependencies.

```bash
pip install -r requirements.txt
```
